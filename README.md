# Gulp + Fractal
![Mud](http://ournameismud.co.uk/css/images/maps-icon.png)

> Frontend starter kit integrated with [Fractal](http://fractal.build/), inspired by [Blendid](https://github.com/vigetlabs/blendid)


###### Requirements
- node >= 8.6.0
- gulp >= 3.91
- yarn/npm
- webpack >= 3.0.0

###### Installation

`yarn add mulp`
